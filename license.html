IntelliJ IDEA 학생 라이선스 받기

IntelliJ IDEA는 무료로 사용 가능한 Community Edition을 배포하고 있지만, Ultimate Edition에 비해 기능상 차이가 많이 납니다.
하지만 학생과 교육자에 한해 Ultimate Edition을 사용할 수 있는 교육 라이선스를 발급해주고 있습니다. 이 라이선스를 발급받는 방법에 대해 알아보겠습니다.

  1. 학생 이메일 필요

    학생 라이선스를 받기 위해서는 우선 학생인증이 된 github 아이디 혹은 학생 이메일이 필요합니다.
    보통 웹메일로 인증이 가능해야하나, 제주대학교는 현재 인증메일이 오지 않는 문제가 있습니다.
    그래서 웹메일 대신 https://portal.jejunu.ac.kr 로 자신의 학번을 이용해 접속한 뒤 상단에 '메일(Google)' 메뉴를 누르면 구글 연동 학생메일을 발급 받습니다.

  2. 학생 라이선스 신청

    아이디가 준비됐다면 이제는 Jetbrains.com에서 신청할 차례입니다. https://www.jetbrains.com/community/education/#students 사이트로 들어가서 For students and teachers에 Apply Now를 누르면 됩니다.

    Github 학생인증 계정이 있다면 Github 로그인을, 웹포털의 이메일을 이용한다면 대학 이메일 주소로 진행하면 됩니다. 정보를 다 입력하고, '무료 제품 신청'을 누르면 완료입니다.

  3. 웹 메일 확인 및 마무리

    정보를 다 입력하고 '무료 제품 신청'을 눌렀다면 웹메일을 확인합니다. 인증메일이 왔다면 메일 안의 Confirm Request를 클릭해 약관을 읽고 동의해주세요.
    이 과정에서 Jetbrains 아이디가 없다면 생성해주고, 로그인을 해보면 학생 인증이 되어있는 것을 확인할 수 있습니다!
    이제 IntelliJ IDEA 다운로드 사이트에서 Ultimate Edition을 받아주고, 아까 만든 Jetbrains 계정으로 로그인하면 완료입니다!
